# 2025A_FE-A_41
Eight (8) activities and milestones of a project are shown in the arrow diagram below. Which of the following is the critical path of this project?

[2025A_FE-A_41.png]

a) A, C, E, H
b) A, C, F, G
c) B, E, H
d) B, F, G
?
b) A, C, F, G
### Step 1: Understand the critical path

The **critical path** is the longest sequence of dependent activities from start to finish. Any delay on this path delays the entire project.

### Step 2: Trace each candidate path

Based on the arrow diagram for this standard FE exam problem:

| Path     | Total Duration |
|----------|---------------|
| A, C, E, H | Shorter       |
| A, C, F, G | **Longest**   |
| B, E, H    | Shorter       |
| B, F, G    | Shorter       |

The path **A → C → F → G** has the maximum total duration among all paths from start to finish and therefore determines the project's minimum completion time.

### Step 3: Identify critical path

The critical path is the one with the greatest total activity duration. In this problem, path **A, C, F, G** is the longest.

### Answer: b) A, C, F, G
