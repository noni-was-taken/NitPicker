# 2025A_FE-A_7
In object-oriented programming, which of the following is the concept that refers to a class that defines a method with the same name and type signature as a method in its superclass?

a) Hiding
b) Inheritance
c) Overloading
d) Overriding
?
d) Overriding
### Explanation

**Overriding** occurs when a subclass provides a new implementation for a method that already exists in its superclass, using the **same name and type signature**. This enables runtime polymorphism. It is distinct from:
- **Overloading**: same method name but different parameter types/count within the same class
- **Inheritance**: the mechanism by which a subclass acquires properties of its superclass
- **Hiding**: refers to field or static method hiding, not instance method redefinition
