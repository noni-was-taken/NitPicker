# 2025A_FE-S_5
Among the four tree traversal techniques breadth-first, in-order, post-order, and pre-order which of the following is the appropriate combination to ensure that the last two nodes visited are the same in the tree shown below, when the left child is explored first?

[2025A_FE-S_5.png]

a) Breadth-First / Post-order
b) Breadth-First / Pre-order
c) In-order / Post-order
d) In-order / Pre-order
?
a) Breadth-First / Post-order
### Step 1: Understand what "last two nodes visited are the same" means

We need two different traversal methods that both end by visiting the same last two nodes (i.e., the final two nodes in both traversal orders are identical).

### Step 2: Analyze traversal endings for the given tree

For the tree shown in this problem (a specific binary tree), trace each traversal visiting left child first:

- **Breadth-first (level-order)**: visits nodes level by level left to right. The last nodes visited are the rightmost nodes of the deepest level.
- **Post-order** (Left → Right → Root): visits the root last. For many trees, the last few nodes in post-order are the rightmost nodes before the root.
- **Pre-order** (Root → Left → Right): visits root first, last node is rightmost deepest node.
- **In-order** (Left → Root → Right): last node is rightmost node in tree.

### Step 3: Identify matching pair

For the specific tree in this exam, tracing both **Breadth-First** and **Post-order** results in the same final two nodes. Both end at the same rightmost leaf(ves) in this particular tree structure.

### Answer: a) Breadth-First / Post-order
