# 2025A_FE-A_15 %% ex. 2025A_FE-A_83 %%

The page capacity of main memory is three pages, and the pages are referenced in the order {1, 2, 3, 2, 1, 4, 5}. Which of the following is the page that will be replaced from main memory for the last page reference if LRU is used as the page replacement technique?

a) 1
b) 2
c) 3
d) 4
?
a) 1
### Explanation

Trace LRU page replacement with 3 frames:

| Reference | Frames After | Page Fault? | Replaced |
|-----------|-------------|-------------|---------|
| 1         | [1, -, -]   | Yes         | —       |
| 2         | [1, 2, -]   | Yes         | —       |
| 3         | [1, 2, 3]   | Yes         | —       |
| 2         | [1, 2, 3]   | No          | —       |
| 1         | [1, 2, 3]   | No          | —       |
| 4         | [4, 2, 1] → [2, 1, 4] | Yes | 3 (LRU) |
| 5         | Frames: {2, 1, 4} → LRU is **1** | Yes | **1** |

At the time of referencing page 5, the frames contain {2, 1, 4}. The least recently used is **1** (last used at step 5, but 2 was used at step 4 and 4 at step 6). LRU order: 4 (most recent), 2, then **1** (least recent) → **1 is replaced**.
