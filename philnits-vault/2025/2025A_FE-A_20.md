# 2025A_FE-A_20
Which of the following is an appropriate description of distributed database?

a) Access to a database server is shared among a globally distributed userbase so that everyone can access the database.
b) It is a database that is distributed to all interested researchers and other users worldwide so that everyone can benefit.
c) It is a NoSQL database instead of a Relational Database Management System (RDBMS).
d) It stores different parts of a database in different locations, and its processing is distributed across those parts.
?
d) It stores different parts of a database in different locations, and its processing is distributed across those parts.
### Explanation

A **distributed database** is a database in which data is physically stored across multiple locations (nodes/sites), but logically appears as a single unified database. Processing is also distributed so that queries can be handled locally where the data resides.

- Option a) describes any shared/network-accessible database, not a distributed one specifically
- Option b) is a description of an open/public dataset, not a technical database architecture
- Option c) incorrectly equates distributed databases with NoSQL; distributed databases can be relational
