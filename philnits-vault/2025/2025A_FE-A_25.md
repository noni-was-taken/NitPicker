# 2025A_FE-A_25 %% ex. 2025A_FE-A_83 %%

Between LAN systems that are different from the transport layer upward in the OSI basic reference model, which of the following is a device that performs protocol conversion?

a) Bridge
b) Gateway
c) Repeater
d) Router
?
b) Gateway
### Explanation

A **Gateway** operates at the highest layers of the OSI model (Layer 4 and above) and performs **protocol conversion** between networks that use entirely different communication protocols. It translates between incompatible systems.

- **Repeater**: operates at Layer 1 (Physical); amplifies signals
- **Bridge**: operates at Layer 2 (Data Link); connects LANs with the same protocol
- **Router**: operates at Layer 3 (Network); routes packets between networks using IP
