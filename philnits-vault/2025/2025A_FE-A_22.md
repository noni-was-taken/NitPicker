# 2025A_FE-A_22
In the context of transaction management, which of the following is a condition that the Isolation property ensures?

a) Data is written permanently after a transaction.
b) Only valid data is written to the database.
c) Transactions are either fully completed or revoked.
d) Transactions are executed without affecting each other.
?
d) Transactions are executed without affecting each other.
### Explanation

The **ACID** properties of transactions are:
- **Atomicity**: transactions are fully completed or fully rolled back (option c)
- **Consistency**: only valid data is written (option b)
- **Isolation**: concurrent transactions execute independently, without interfering with each other (option d) ✅
- **Durability**: committed data is written permanently (option a)

Each option corresponds to a different ACID property; **Isolation** = option d.
