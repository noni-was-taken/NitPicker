# 2025A_FE-S_13
The relation between virtual address and physical address is shown below. Each virtual page and page frame starts at a multiple of 4096 and ends 4095 addresses higher. Virtual address space 4K-8K corresponds to 4096-8191 and 8K to 12K corresponds to 8192-12287. Which of the following is the physical address corresponding to the virtual address numbered 8300?

[2025A_FE-S_13.png]

a) 8192
b) 8300
c) 24576
d) 24684
?
d) 24684
### Step 1: Find the virtual page number and offset

Virtual address = 8300

Page size = 4096 bytes

$$\text{Virtual page number} = \left\lfloor \frac{8300}{4096} \right\rfloor = \left\lfloor 2.026... \right\rfloor = 2$$

$$\text{Page offset} = 8300 \mod 4096 = 8300 - 8192 = 108$$

So virtual address 8300 is in **virtual page 2**, with offset **108**.

### Step 2: Look up the page table

Virtual address space 8K–12K (virtual page 2) maps to a physical page frame. From the diagram in the problem, virtual page 2 maps to **physical frame 6**.

Physical frame 6 starts at:
$$6 \times 4096 = 24576$$

### Step 3: Compute physical address

$$\text{Physical address} = 24576 + 108 = 24684$$

### Answer: d) 24684
