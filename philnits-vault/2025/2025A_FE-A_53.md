# 2025A_FE-A_53
Basic elements of IoT in a factory's equipment maintenance tasks: Data collection → A; Data transmission → B; Data analysis → C; Data utilization → D. Which of the following corresponds to A (Data collection)?

a) Abnormal value judgment tool
b) Display for work instructions
c) Temperature sensor for equipment
d) Wireless communication within the factory
?
c) Temperature sensor for equipment
### Explanation

In an **IoT workflow**, the stages map as follows:

| Stage | Role | Example |
|-------|------|---------|
| **A - Data collection** | Physically gathering raw data from the environment | **Temperature sensor** ✅ |
| **B - Data transmission** | Sending data to processing systems | Wireless communication |
| **C - Data analysis** | Processing/interpreting the data | Abnormal value judgment tool |
| **D - Data utilization** | Acting on analysis results | Display for work instructions |

A **temperature sensor** collects raw physical data from equipment — it is the first step in the IoT pipeline.
