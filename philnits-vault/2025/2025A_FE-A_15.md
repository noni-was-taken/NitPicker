# 2025A_FE-A_15
The page capacity of main memory is three pages, and the pages are referenced in the order {1, 2, 3, 2, 1, 4, 5}. Which of the following is the page that will be replaced from main memory for the last page reference if LRU is used as the page replacement technique?

a) 1
b) 2
c) 3
d) 4
?
b) 2
### Explanation

Trace LRU page replacement with 3 frames:

| Reference | Frames After | Page Fault? | Replaced |
|-----------|-------------|-------------|---------|
| 1         | [1, -, -]   | Yes         | —       |
| 2         | [1, 2, -]   | Yes         | —       |
| 3         | [1, 2, 3]   | Yes         | —       |
| 2         | [1, 2, 3]   | No          | —       |
| 1         | [1, 2, 3]   | No          | —       |
| 4         | [1, 4, 2] → replaces 3 | Yes | 3 (LRU) |
| 5         | Frames: {1, 4, 2} → LRU is **2** | Yes | **2** |

At the time of referencing page 5, the frames contain {1, 4, 2}. The least recently used is **2** (last used at step 4), while 1 was used at step 5 and 4 was used at step 6. LRU order: 4 (most recent), 1, then **2** (least recent) → **2 is replaced**.
