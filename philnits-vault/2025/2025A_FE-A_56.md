# 2025A_FE-A_56 %% ex. 2025A_FE-A_83 %%

Which of the following is an example of Green IT?

a) Development of a powerful in-house data center
b) Development of efficient in-house software
c) Using a desktop PC instead of a thin client laptop
d) Using web conferencing instead of traveling to meetings
?
d) Using web conferencing instead of traveling to meetings
### Explanation

**Green IT** refers to the practice of using computing resources in an environmentally sustainable and energy-efficient way. Using **web conferencing** reduces the need for business travel, thereby lowering carbon emissions.

- Option a) building a more powerful data center typically *increases* energy consumption
- Option b) software efficiency is useful but not directly a Green IT practice in the typical environmental sense
- Option c) desktop PCs generally consume *more* power than thin clients — the opposite of Green IT
