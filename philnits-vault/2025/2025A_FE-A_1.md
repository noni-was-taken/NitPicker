# 2025A_FE-A_1
In the single precision format of the 'IEEE Standard for Binary Floating-Point Arithmetic' (IEEE 754), a 32-bit floating point number is represented as below: [S: Sign (1 bit), E: Exponent (8 bits), F: Fraction (23 bits)]. If $0<E<255$, then the value of the number is $(-1)^{S}\times2^{(E-127)}\times(1+F)$ where F represents the fractional part of the number, stored as a 23-bit binary sequence. This F is the sum of its weighted binary components: $F=b_{1}\times2^{-1}+b_{2}\times2^{-2}+\cdot\cdot\cdot+b_{23}\times2^{-23}$, where $b_1, b_2,...,b_{23}$ are the binary digits of F. According to the IEEE-754 standard, which of the following is the decimal equivalent of the 32-bit floating point number given below?

`01000001011000000000000000000000`

a) 0.1875
b) 0.4375
c) 6.0
d) 14.0
?
d) 14.0
### Explanation

Break the 32-bit string into its three fields:
- **S** = `0` → positive number
- **E** = `10000010` = 130 in decimal
- **F** = `11000000000000000000000`

Calculate the value:
- Exponent: $2^{(130 - 127)} = 2^3 = 8$
- Fraction: $F = 1 \times 2^{-1} + 1 \times 2^{-2} = 0.5 + 0.25 = 0.75$, so $(1 + F) = 1.75$
- Result: $(-1)^0 \times 8 \times 1.75 = \mathbf{14.0}$
