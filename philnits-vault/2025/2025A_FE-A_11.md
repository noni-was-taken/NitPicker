Which of the following is the most appropriate explanation of edge computing?

a) Increasing the operability and expressiveness of web application software by performing screen creation and data processing on the client side
b) Increasing the real-time nature of processing by performing primary processing of data on a server or other such computer in a location close to the origin of data, such as a sensor that is attached to a user or a device
c) Increasing throughput with parallel processing by virtualizing and integrating several servers and PCs and creating a single high-performance computer
d) Starting up the required server only when data comes in, and freeing up resources by stopping the server as soon as the processing is finished

?
b) Increasing the real-time nature of processing by performing primary processing of data on a server or other such computer in a location close to the origin of data, such as a sensor that is attached to a user or a device
### Explanation

**Edge computing** moves data processing closer to the data source (e.g., IoT sensors, local devices) rather than sending everything to a central cloud. This reduces latency and improves real-time responsiveness.

- Option a) describes **rich client / fat client** computing
- Option c) describes **grid or cluster computing**
- Option d) describes **serverless / FaaS (Function as a Service)** computing
