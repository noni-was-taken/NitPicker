# 2025A_FE-A_28 %% ex. 2025A_FE-A_83 %%

When X receives a message with a digital signature that uses public key cryptography from Y, which of the following is the key that X uses to verify that the message truly came from Y?

a) X's private key
b) X's public key
c) Y's private key
d) Y's public key
?
d) Y's public key
### Explanation

In **digital signature** verification using public key cryptography:
1. The **sender (Y)** signs the message using **Y's private key**
2. The **receiver (X)** verifies the signature using **Y's public key**

Since Y's public key is openly available, anyone (including X) can verify that the message was indeed signed by Y (the only one who has Y's private key). This provides **authentication and non-repudiation**.
