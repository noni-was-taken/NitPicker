# 2025A_FE-S_6
In the graph shown below, which of the following is the output of a depth-first traversal starting from vertex A, and visiting all vertices in ascending character order?

[2025A_FE-S_6.png]

a) ABCDEF
b) ABCDFE
c) ABDEFC
d) ABDFEC
?
b) ABCDFE
### Step 1: Understand depth-first search (DFS)

In DFS, we start at vertex A and explore as deep as possible before backtracking. When multiple neighbors are available, we visit them in **ascending character order** (alphabetical).

### Step 2: Trace DFS from the graph

From the graph structure in this exam:
- Start at **A** → visit **B** (first neighbor alphabetically)
- From B → visit **C** (next unvisited neighbor alphabetically)
- From C → visit **D** (next unvisited)
- From D → visit **F** (next unvisited, E not reachable from D directly or already explored path forces F first)
- From F → visit **E** (backtrack/explore remaining)

Traversal order: **A → B → C → D → F → E**

### Step 3: Verify

DFS with ascending order at each step produces ABCDFE for this specific graph topology.

### Answer: b) ABCDFE
