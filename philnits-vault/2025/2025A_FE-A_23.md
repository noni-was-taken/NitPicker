# 2025A_FE-A_23
Which of the following is a mechanism used to prioritize traffic based on its importance and minimize network latency for network applications?

a) AAA
b) PoE
c) QoS
d) RTP
?
c) QoS
### Explanation

**QoS (Quality of Service)** is a set of technologies and mechanisms used to manage network traffic by prioritizing certain types of data (e.g., voice, video) over others, reducing latency, jitter, and packet loss for critical applications.

- **AAA** (Authentication, Authorization, Accounting): security framework for network access control
- **PoE** (Power over Ethernet): delivers electrical power via network cables to devices like IP phones and cameras
- **RTP** (Real-time Transport Protocol): a protocol for transmitting real-time audio/video, but not a QoS mechanism itself
