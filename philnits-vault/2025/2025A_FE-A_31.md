# 2025A_FE-A_31 %% ex. 2025A_FE-A_83 %%

Which of the following is a function of an IDS?

a) It checks whether software products that are installed on a PC are the latest version.
b) It collects the status of information security measures, such as the status of operations management for an information system...
c) It monitors a server or a network and notifies the administrator when it has detected an intrusion or infringement.
d) It sends test data to a product that it examines and detects any vulnerability from the response or behavior of the product.
?
c) It monitors a server or a network and notifies the administrator when it has detected an intrusion or infringement.
### Explanation

An **IDS (Intrusion Detection System)** passively monitors network traffic and system activity for signs of unauthorized access, policy violations, or attacks, then **alerts** administrators.

- Option a) describes a **patch management** or **software update** tool
- Option b) describes a **SIEM (Security Information and Event Management)** system
- Option d) describes a **vulnerability scanner** or **fuzzing tool**
