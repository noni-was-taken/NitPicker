# 2025A_FE-A_26 %% ex. 2025A_FE-A_83 %%

Which of the following is an Internet standard that extends the format of email header fields to handle not only text messages but also audio and image files?

a) HTML
b) MHS
c) MIME
d) SMTP
?
c) MIME
### Explanation

**MIME (Multipurpose Internet Mail Extensions)** extends the basic email format (originally limited to ASCII text) to support multimedia content such as images, audio, video, and binary files. It defines how non-text attachments are encoded and identified in email headers.

- **HTML**: a markup language for web pages, not an email extension standard
- **MHS** (Message Handling System): an older X.400-based messaging standard
- **SMTP** (Simple Mail Transfer Protocol): the protocol for *sending* email, not for handling attachments
