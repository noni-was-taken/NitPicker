# 2025A_FE-A_29 %% ex. 2025A_FE-A_83 %%

The IP address of server X that is owned by an attacker, is recorded on Company B's DNS cache server as the IP address corresponding to the FQDN of Company A's web server. As a result, which of the following are the users that unintentionally connect to server X?

a) Company A employees who try to connect to Company A's web server
b) Company A employees who try to connect to Company B's web server
c) Company B employees who try to connect to Company A's web server
d) Company B employees who try to connect to Company B's web server
?
c) Company B employees who try to connect to Company A's web server
### Explanation

**DNS cache poisoning** has been performed on **Company B's DNS cache server**. The poisoned entry maps Company A's domain name (FQDN) to the attacker's server X IP address.

When **Company B's employees** try to resolve Company A's domain, they query **Company B's DNS cache server**, which returns the attacker's IP → they connect to server X instead of Company A's real server.

Company A's employees use **their own DNS server**, which is not poisoned, so they are unaffected.
